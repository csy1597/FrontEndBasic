# Array 란
    - 하나의 변수 안에 많은 데이터를 저장할 때 사용한다. 
    - 배열 변수 안에 데이터 유형은 상관이 없으나 가급적이면 데이터 유형은 통일하여 저장하는 것이 좋다.
